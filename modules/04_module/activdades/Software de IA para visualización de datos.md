# Actividad 7 | Investigación: Software de IA para visualización de datos

## Objetivo
3.1 Construir  visualizaciones de datos para dar respuesta a las preguntas del negocio.

## Instrucciones
A través de esta actividad vas a ampliar tus conocimientos sobre las principales herramientas de IA para visualización de datos, así como entender mejor las funcionalidades que ofrecen para crear dashboards y narrativas visuales, además vas a evaluar la aplicabilidad de estas herramientas en diferentes contextos profesionales o académicos. Para lograrlo sigue estos pasos:

1. Selecciona al menos tres softwares o plataformas de IA que se especialicen en visualización de datos y creación de dashboards. Pueden ser comerciales, de código abierto o basadas en la nube.  
2. Para cada herramienta, describe:  
    - Nombre y proveedor o desarrollador.  Anexa el link del sitio.
    - Funcionalidades principales relacionadas con la visualización, dashboards y storytelling. 
    - Capacidades de integración con datos y automatización. 
    - Ejemplos de casos de uso o aplicaciones prácticas.  
    - Compara brevemente las herramientas en aspectos como facilidad de uso, capacidades de storytelling, personalización y costo.  
3. Concluye con una reflexión sobre las ventajas, limitaciones y potenciales aplicaciones de estas tecnologías. Justifica 4. tu conclusión apoyada en el proceso del análisis de datos. Evalúa el impacto de incluir estas herramientas en las organizaciones.

## Especificaciones de entrega
Formato: Archivo PDF con portada que incluya nombre y matrícula del alumno y de 2 a 3 páginas.
